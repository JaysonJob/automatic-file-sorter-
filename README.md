# automatic-file-sorter-

This project is a simple Python automation script that organizes files into folders based on their file type in the file explorer.

## Features

- Moves `.csv` files into a **csv files** folder
- Moves `.png` image files into an **image files** folder
- Moves `.txt` files into a **text files** folder


## Technologies Used

- Python
- os module
- shutil module
